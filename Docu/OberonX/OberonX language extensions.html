OberonX language extensions

The OberonX language extensions are one result of the Oberon-XSC (eXtensions for Scientific Computing) project of Karlsruhe University and ETH.

Publications and Links

  • R. Morelli, Diploma Thesis: Integration of OberonX into Oberon, Winter 1996
  • P. Januschke, Dissertation: Oberon-XSC, Eine Programmiersprache und Arithmetikbibliothek für das Wissenschaftliche Rechnen, Universität Karlsruhe 1998
  • XSC-Languages page at the Institut für Angewandte Mathematik in Karlsruhe.
  • Chapter7.Text in the Native Oberon Release. -

Language extensions

  • structured return types
  • operators
  • new array types

Availability

The OberonX language extensions are available in the Native Oberon System from release 2.2.3.
